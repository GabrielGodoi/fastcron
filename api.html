Types

struct FastCron_t

Bitmask-encoded cron schedule.

Each field stores one bit per valid unit value:

  • minutes — bits 0..59 (60 bits of a uint64_t)

  • hours — bits 0..23 (24 bits of a uint32_t)

  • days_of_month — bits 1..31 (bit 0 unused)

  • months — bits 1..12 (bit 0 unused)

  • days_of_week — bits 0..6 (Sunday = 0)

Core Functions

time_t fastcron_get_next_wakeup(const FastCron_t *mask, time_t current_epoch)

Compute the next matching epoch from a bitmask schedule.

Uses O(1) bit-scan intrinsics per field. Pure math epoch conversion with no OS/timezone dependencies.

Parameters:

  • mask – [in] Pre-populated bitmask schedule.

  • current_epoch – [in] Current UNIX timestamp (UTC).

Returns:

Next matching epoch (always > current_epoch), or (time_t)-1 on error.
